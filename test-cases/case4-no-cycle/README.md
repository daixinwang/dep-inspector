# 测试用例 4：无循环
## 项目结构

```
src/
├── A.ts (imports B)
├── B.ts (imports C)
└── C.ts (无导入)
```

## 依赖关系

```
A → B → C
```

这是一个有向无环图 (DAG)。

## 预期结果

- **节点数**：3
- **边数**：2
- **循环数**：0
- **循环路径**：无

## 说明

这是一个正常的、健康的依赖结构，没有任何循环依赖。
这种结构可以进行拓扑排序，依赖关系清晰明确。
